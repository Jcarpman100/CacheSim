Howdy!

Compile command 
g++ cachesimulator.cpp -o cachesimulator

run command
./cachesimulator.(exe/out) <input filename>

The (.exe/.out) will depend on the system it is compiled on 

Example
$ g++ cachesimulator.cpp -o cachesimulator
$ ./cachesimulator.exe input.txt
